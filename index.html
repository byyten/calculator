The Internal Debate

(the infernal debate)

the brief for this project suggested building a simple calculator that evaluated a pair of numbers and display a result before continuing to evaluate any further inputs which seemed in my mind to be both correct and mathematically incorrect because if the whole expression were to be evaluated using mathematical precedence the result would often be different to the brief's suggested implementation and the brief's suggested implementation would at times be wrong (mathematically).

that creates a disonance in my mind and thus led me to abandon the suggestion and evaluate the input expressions using operator precedence in a more correct interpretation of the math

this behaviour also gets me into trouble as an employee because I don't necessarily follow directions and then be accused of not following instructions, the project objectives, being obstructive, lazy, disobedient, ... so, just don't it ok! never mind that the solution might be better for that effort
